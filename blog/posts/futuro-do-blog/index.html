Atualizações no Blog - Abril

Andei pensando sobre possíveis atualizações futuras no blog. Alguns dias atrás pensei em implementar um back end utilizando a replit, já que o GitHub Pages não permite back end naturalmente. Daí era só implementar um iframe num index.html e pronto \o/, o site estaria brilhando. Mas a ideia parece ser uma gambiarra muito grande, então não sei se vou continuar com ela.

O que desejo fazer no blog no futuro é:

  • Transformar os arquivos HTML das postagens em markdowns, facilitando a minha e a sua vida (quero fazer o quanto antes)
  • Categorizar os posts, com um índice para cada categoria
  • Tomar vergonha na cara e priorizar o desenvolvimento do front end no índice do blog
